On your iPhone, a Shortcut is a quick way of changing settings and interacting with apps. For this next step in the study, we will ask you to install and run a Shortcut. This Shortcut will the default search engine on your iPhone.

Please follow the steps below to install and run the required Shortcut on your iPhone.

  1. Tap this link to access the Shortcut.

  2. If an Add Shortcut button appears, tap it to install the Shortcut. Do not run the Shortcut after installing it. Simply install it and return to this page.

    If a Get Shortcut button appears in Safari, you will need to install the Apple Shortcuts app. Tap the button and follow the prompts to install the app, then begin these instructions again.

  3. Once the Shortcut is installed, tap this link this link to run it.

Once you have completed running the Shortcut, it will automatically direct you to survey questions about your experience with the Shortcut. If you are not redirected to the survey, please re-run the Shortcut and ensure you follow it through to completion. Once you reach the survey, you may close this page if it remains open.
